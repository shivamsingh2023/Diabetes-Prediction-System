# Diabetes Prediction System

**Diabetes Prediction System** is a machine learning project developed in Python and deployed using Streamlit. It predicts whether a patient is diabetic based on health parameters.

## Features
- Predict diabetes using Random Forest model
- Handles feature scaling and preprocessing
- Shows predictions through an interactive Streamlit app
- Evaluation using metrics like Accuracy and ROC-AUC
